Debug endpoints

These endpoints are only active under development.

  • /configuration

    Show current configuration as JSON

  • /generate-reports-now

    Trigger report generation. Optional query parameter 'day' where you can fill in the day you wish to make a report of in the format YYY-MM-DD.

  • /dump

    Dump the data contained in the write cache as a TTL file. Optional query parameter 'filename' where you can input the file name of the dump file. Do not use an extention.

  • /clear-store

    Clear the write cache where the last reports are cached

  • /dump-files

    Show index of the created dump files

  • /dump-files/filename

    Download specific dump file
